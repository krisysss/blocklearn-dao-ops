# User Onboarding Guide

## Wallet Installation
1. Visit the official website: www.blocklearndao.com
2. Click "Download" and select the version for your operating system
3. Ensure your internet connection is stable before installation

## Join the Blockchain Network
1. Go to the website and select "Register Node"
2. You will receive a 12-word recovery phrase
3. Keep this phrase secure and do not share it with anyone
4. Use the recovery phrase to join the test network for voting and testing

## Important Note
This project operates on a testnet. Do not use real assets or real data.
